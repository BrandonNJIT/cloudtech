What is virtualization? What is containerization?

Cloud: Virtualization is the running of an application or program through a virtual machine or a separate instance of an emulated computer through another machine. Containerization is the running of a process on a linux operating system. While very similar, the container is running the operation on the same linux kernel as the hostâ€™s operating system. The differences between the two is that the virtualization is basically a different application from the operating systems application kernel. With containerization the applications are running on the same layer of operations as the rest of the applications running on linux.
